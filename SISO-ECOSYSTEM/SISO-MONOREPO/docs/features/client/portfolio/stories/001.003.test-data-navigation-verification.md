## Acceptance Criteria

### Data Population (AC 1-5)

1. **AC1**: 4 total complete client examples exist (Mayorker + 3 new clients)
2. **AC2**: New clients represent diverse industries: Restaurants, Crypto, and one additional industry (Agency or eCommerce)
3. **AC3**: Each new client has all required data fields populated (following Mayorker structure):
   - Basic info: id, name, tagline, description, type, status
   - Pricing: min, max, currency, deliveryTime
   - Features: key features (5+), technical features (5+), integrations (3+)
   - Tech stack: frontend, backend, database, hosting, tools
   - Timeline: 4+ phases with names, descriptions, durations
   - Media: logo path, screenshot placeholders (3+ desktop, 3+ mobile)
   - Industry, testimonial (optional), metadata

4. **AC4**: All client data files export correctly from `src/domain/portfolio/data/index.ts`
5. **AC5**: Placeholder images are referenced correctly (actual images not required - placeholders OK)

### Navigation Flow Testing (AC 6-15)

6. **AC6**: Complete user journey works: Home → Portfolio Hub → Industry Landing → Client Detail
7. **AC7**: Breadcrumb navigation works in all directions (Portfolio ← Industry ← Client)
8. **AC8**: Industry filter on PortfolioHub correctly routes to industry pages
9. **AC9**: Project cards on IndustryLanding correctly route to client detail pages
10. **AC10**: "View Live Site" buttons on ClientDetail pages open correct URLs (or show disabled state)
11. **AC11**: Related projects section links navigate correctly
12. **AC12**: Back navigation works (browser back button preserves state)
13. **AC13**: Direct URL navigation works for all routes:
    - `/portfolio`
    - `/portfolio/restaurants`
    - `/portfolio/restaurants/restaurant-client`
14. **AC14**: 404 pages display correctly for invalid slugs
15. **AC15**: All tab switches on ClientDetail pages work smoothly (no console errors)

### Quality Verification (AC 16-22)

16. **AC16**: No TypeScript compilation errors
17. **AC17**: No console errors or warnings on any page
18. **AC18**: All pages load in < 2 seconds (measured with Chrome DevTools)
19. **AC19**: Mobile experience verified on 3 screen sizes (375px, 768px, 1920px)
20. **AC20**: Accessibility tested with keyboard navigation (Tab, Enter, Arrow keys)
21. **AC21**: Lighthouse scores > 90 for Performance, Accessibility, SEO (all 3 pages)
22. **AC22**: Cross-browser compatibility verified (Chrome, Firefox, Safari minimum)
